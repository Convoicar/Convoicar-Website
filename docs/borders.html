Borders

Landkit only

Use border utilities to quickly style the border and border-radius of an element. Great for images, buttons, or any other element.

Colors

Additonal border color utilities. Additonal modifier at the end of the utility name means the opacity level.

<div class="mr-3 mb-3 border border-white-10" style="width: 5rem; height: 5rem;"></div>
<div class="mr-3 mb-3 border border-white-20" style="width: 5rem; height: 5rem;"></div>
<div class="mr-3 mb-3 border border-gray-300" style="width: 5rem; height: 5rem;"></div>
<div class="mr-3 mb-3 border border-gray-800" style="width: 5rem; height: 5rem;"></div>
<div class="mr-3 mb-3 border border-gray-800-50" style="width: 5rem; height: 5rem;"></div>
<div class="mr-3 mb-3 border border-gray-900-50" style="width: 5rem; height: 5rem;"></div>
